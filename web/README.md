## ⚡ 开发依赖安装说明

基础模块：

```bash
# 自动格式化代码
npm i --save-dev --save-exact prettier
npm i --save-dev @types/node 
```

开发安装：

```bash
# 字体
npm i misans

# UI 和图标
npm i antd@4.24.14
npm i @ant-design/icons
npm i moment

# 路由
npm i react-router
npm i react-router-dom

# 样式
npm i less less-loader

# 网络请求
npm i axios

# 数据
npm i valtio
```
